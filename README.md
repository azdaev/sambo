# Sambo Borz Team

Сайт спортивной секции самбо.

## Установка

```bash
npm install
cd server && npm install
```

## Запуск

### Разработка

```bash
# Сервер
cd server && npm run dev

# Фронтенд
npm run dev
```

### Продакшн

```bash
# Сборка фронтенда
npm run build

# Запуск сервера
cd server && npm start
```
